# ASCII Value Notes

## Introduction
ASCII (American Standard Code for Information Interchange) is a character encoding standard used in computers and electronic devices. ASCII codes represent text in computers, communications equipment, and other devices that use text. Each character (letter, number, or symbol) is assigned a unique 7-bit binary number.

## Additional Notes
- The ASCII table consists of 128 characters, ranging from 0 to 127.
- The first 32 characters (0-31) are non-printable control characters.
- Characters from 32 to 126 are printable characters, including letters, digits, punctuation marks, and a few miscellaneous symbols.
- Character 127 is the "delete" control character.
